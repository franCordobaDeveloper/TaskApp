
# TaskApp

Proyecto desarrollado para el challenge de ingreso a Academia ForIt 2025.  
Aplicación para gestionar tareas con backend en Node.js y frontend en React.

---

## Requisitos previos

- Node.js (versión 16 o superior recomendada)  
- npm o yarn  
- Git
